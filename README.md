## About Me

I'm **Anuja Gaikwad**, an MSc Data Science graduate from **Coventry University** with a strong passion for turning raw data into actionable insights. My dissertation — **Protein-to-Protein Interaction Analysis for Cancer Prediction** — used graph-based machine learning to classify cancer-associated proteins with **~89% accuracy**. I'm actively seeking **Data Analyst / Data Scientist roles in the UK** and am fully eligible to work here.

- MSc Data Science (Distinction) — Coventry University (2025)
- Built 6 end-to-end data projects spanning healthcare, retail, education and public policy
- Retail pricing pipeline analyzed 9M+ product listings across 5 UK supermarket chains
- Cancer protein classifier achieved ~89% accuracy using network science + XGBoost
- Fully eligible to work in the UK
